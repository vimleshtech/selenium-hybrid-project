Maven			: is configuration and build management tool
			:
			 Group ID   : set of application just like solution 
			 Artifect ID: is application name 

			 pom.xml: configuration(project name, version) and dependency (.jar or .war file)
			 Project Structure
				src   : source code 
				test  : test script
				target: output folder
			
			Maven life cycle:
				validate  -> clean ->  compile  -> test  -> package  -> install
					     clear- remove previous bild or unrequired files
					     compile - code compilation 
					     test - test script execution 
					     package - create a build (.jar,.war file)
